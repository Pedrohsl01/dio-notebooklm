# Aprendizado Ativo com NotebookLM: Educação Financeira 📈

Este repositório contém o resultado do projeto prático realizado na plataforma DIO, focado em utilizar o **NotebookLM** como ferramenta de apoio ao estudo e organização de conhecimento.

## 🎯 Objetivo
O objetivo foi criar um caderno temático sobre **Educação Financeira**, utilizando a Inteligência Artificial para processar fontes externas e gerar materiais de estudo estruturados.

## 🛠️ O que foi desenvolvido:
* **Curadoria de Fontes:** Seleção de 10 fontes abertas (artigos, PDFs e vídeos) sobre finanças.
* **Miniguia de Estudos:** Resumos estruturados gerados pela IA com base nas fontes.
* **Glossário de Conceitos:** Definições claras de termos do mercado financeiro.
* **Prompt Engineering:** Testes e refinamentos de comandos para obter respostas mais precisas da ferramenta.

## 🚀 Como acessar
Você pode visualizar o miniguia final e os prompts utilizados na pasta `/docs` deste repositório.
